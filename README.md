# Stock Market Analysis

## Project Overview

The Stock Market Trend Prediction project analyzes historical stock prices and predicts whether the stock price will increase or decrease the next day. This project demonstrates time-series analysis, feature preprocessing, and trend prediction using pandas and numpy only.

It uses a simple moving average crossover strategy to predict the trend, making it suitable for beginner and intermediate-level projects.
